1. Project Overview
   Smart Wheelchair is a graduation project that provides a complete intelligent control and monitoring system for wheelchairs. The system integrates face recognition authentication, real-time GPS navigation,   live sensor data streaming, and dual control modes (Manual & Autonomous) to enhance mobility and independence for people with disabilities.
   1.1 System Goals
•	Provide secure face-recognition-based authentication with liveness detection (eye blink verification)
•	Enable real-time wheelchair monitoring via a modern web dashboard
•	Support dual control modes: Manual joystick and Autonomous navigation
•	Allow campus navigation by selecting pre-mapped faculty destinations
•	Stream live sensor data (position, velocity, battery, accuracy) via WebSocket
•	Log attendance and obstacle detection events to a cloud database
